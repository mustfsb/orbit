# Orbit Product Vision

## North Star

Orbit aims to be a calm personal operating surface for planning, focus, and reflection. The product should help a single user decide what matters, stay with it, and see the relationship between daily actions and longer-term goals without scattering attention across separate tools.

## Product Promise

- Turn planning into an actionable weekly rhythm instead of a static backlog.
- Keep focus sessions, tasks, goals, journal notes, and review surfaces connected enough to feel like one system.
- Use AI where it reduces planning friction, not where it adds ceremony.
- Preserve a quiet interface that feels intentional rather than overloaded.

## Core User Loop

1. Capture and shape work into a believable plan.
2. Execute with a focused timer and a clear next action.
3. Track momentum through goals, analytics, and recent history.
4. Reflect through journal notes and adjust the next plan.

## Product Principles

- Calm by default: the interface should reduce noise, not amplify it.
- One personal workspace: planning, focus, goals, journal, analytics, and library should feel connected.
- Durable progress: important user work should increasingly survive across sessions and devices.
- Helpful intelligence: planner and future AI features should operate on product context, not generic prompts alone.
- Honest scope: Orbit should expand from a strong single-user core before taking on broad integrations or collaboration.

## Near-to-Mid-Term Direction

- Strengthen the single-user system before adding major new surface area.
- Make the planner and library feel as durable as the synced task, goal, session, and journal domains.
- Build a better context layer so AI planning can use real product data more reliably.
- Add carefully chosen integrations only after the core planning loop is stable.
